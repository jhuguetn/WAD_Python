Tools.Input2DCM.Input2DCM.sendasdicom(dcmconfig, datastr)[source]¶
create a dicom file with the datastr embedded as a private tag, and send it to dcm4chee
dcmconfig is a dict that should at least contain: dest_aet, src_aet, dest_ip, dest_port datastr is a string that should be of even string length (required for private tags content)

returns output of create_dicom: (Succes,msg) with Succes True or False and msg a string or None
